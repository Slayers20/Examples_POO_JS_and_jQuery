Introducción a la POO

Programación Orientada a Objetos

Que es la POO

Es un paradigma de programación basado en técnicas como el encapsulamiento, la herencia, la moularidad...

Encapsulamiento

No tienes por que saber como funciona por dentro.

var a = new Array(); //tu no sabes lo que hace ahí dentro (o si..)

Modulariad

Códigos independientes y reaprovechamiento del mismo.

RegistroUsuario(nombre, email);

Herencia

Características que pasan de padres a hijos

Persona (padre) Hombre, Mujer, Jovencito (hijos)

Que son las clases y los objetos

Conjunto de atributos y métodos que definen las características y propiedades de nuestro objeto, dan una identidad al objeto.

Clases: Coche, Moto, Bici

Objetos: MiBiciNueva, CocheVecino

Que son los atributos

Características que tienen las clases, estas características se aplicarán a los objetos.

Tener 4 ruedas, Ser de color Verde, etc.

Que son los métodos

Propiedades que tienen las clases, con las que podemos trabajar para modificar los objetos o conocer atributos de estos.

Saber las ruedas que tengo, quitar una rueda, pintar de otro color, etc.

Ejemplo POO con Javascript

                
//creo una nueva figura
var caja = new figura("red", "Hola");

//Miro que tono tiene
console.log(caja.getTono());

//cambio un Tono
caja.setTono("blue");

//duplico la figura
var caja2 = caja.duplicate();

//cambio un contenido
caja2.setText("Adios");

//creo un circulo (¡¡HERENCIA!!)
var redonda = new circulo(300);

Ejemplo ""POO"" con jQuery

                
//creo una nueva figura
var caja = $("section").figura("caja", "red", "Hola");

//Miro que Tono tiene
console.log(caja.getTono());

//cambio un Tono
caja.setTono("blue");

//duplico la figura
var caja2 = caja.duplicate("caja2");

//cambio un contenido
caja2.setText("Adios");

//creo un circulo (¡¡HERENCIA!!)
var redonda = $("section").circulo("redonda",300);
